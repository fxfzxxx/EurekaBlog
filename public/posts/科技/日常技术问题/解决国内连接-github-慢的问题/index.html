在没有科学上网的情况下, 经常会遇到 GitHub 连接失败的情况. 经查发现, 可以通过配置 /private/etc/hosts 可以达到不搭梯子仍可快速访问 GitHub 的效果.

首先到 https://websites.ipaddress.com/ 下查询 GitHub 及其子域名的最新 ip 地址.

打开命令行输入以下命令:
