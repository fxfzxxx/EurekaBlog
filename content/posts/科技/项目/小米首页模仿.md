### 项目的特点

1. CSS 实现的下拉菜单, 并且鼠标悬浮菜单不收回.
1. CSS 实现了众多动态效果, 例如首页图表的雪碧图动画.
2. 利用爬虫脚本得到的 JSON 格式小米商品数据, 运用原生 JS 将数据动态填充至菜单页中.
3. 原生 JS 实现的轮播图.

### 项目的技术栈

- html, CSS, JS
